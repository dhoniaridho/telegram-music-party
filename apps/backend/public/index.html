How to Install a Chrome Extension from ZIP

Download ZIP

  1. Extract the ZIP file if it is compressed.
  2. Open Google Chrome.
  3. Go to chrome://extensions/ in the address bar.
  4. Enable Developer mode (toggle in the top-right corner).
  5. Click "Load unpacked".
  6. Select the extracted folder containing the extension.
  7. The extension will be installed and ready to use.

How to Use this Bot

  1. Add @xmsc_bot to your Telegram group.
  2. Run the command /register in the group chat.
  3. Input the code displayed in the extension to link the extension to the bot.
  4. Click the "Join" button in the extension to join the group chat.

How to Add Queue

  1. Find the music you want to add to the queue.
  2. Tag @xmsc_bot in the group chat, followed by "search [music name] here" (e.g. @xmsc_bot search Shape of You here).
  3. Bot will display the results of the search. Click the "Add to queue" button next to the desired result.
  4. Then, run the command /play in the group chat to play the music from the queue.
